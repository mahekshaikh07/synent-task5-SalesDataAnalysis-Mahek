# Task 5: Sales Data Analysis

## Problem Statement
Analyze Superstore sales data to uncover business performance trends, top products, and profit insights.

## Dataset
- Source: [Kaggle - Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
- Rows: 9994 | Columns: 21

## Approach
1. Loaded and preprocessed the dataset
2. Extracted time features from Order Date
3. Analyzed monthly revenue trends across years
4. Identified top-selling products
5. Compared sales vs profit across categories and sub-categories
6. Analyzed regional performance

## Key Insights
- Sales spike every Nov-Dec (holiday season)
- Technology is most profitable; Furniture has poor margins
- Tables and Bookcases are loss-making sub-categories
- West region leads in sales
